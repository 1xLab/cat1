# App – Laboratórios e Parceiros

## Contexto

Página de login para laboratório/parceiro, voltada ao acesso a funcionalidades específicas (por exemplo, cadastro de coletas, acompanhamento de exames, faturamento).

## Título

- **JM BioAnalises | Login**

## Conteúdo atual

Elementos presentes:

- Texto: **Não tem uma conta? Inscrever-se**  
- Foco do público: **LABORATÓRIO E PARCEIROS**  
- Link: **Esqueceu sua senha?**  
- Botão: **Fazer Login**  
- Link auxiliar: **Voltar**

## Estrutura sugerida para o dev

- Campo de usuário (e-mail corporativo ou identificador do laboratório).  
- Campo de senha.  
- Botão **Fazer Login**.  
- Link **Esqueceu sua senha?** com fluxo de recuperação.  
- Link **Inscrever-se** apontando para cadastro de parceiros/laboratórios.  
- Link **Voltar** retornando ao site principal ou dashboard anterior.
