# Política de Privacidade – JM Tox

## Cabeçalho / navegação

- Dúvidas  
- Postos de Coletas  
- Parceiros  
- Contato  
- Resultado  
- Acesso Restrito / Portal do Parceiro

## Quem somos

- O endereço do site é apresentado como: `/`.  
- A política aplica-se aos dados coletados pelo site da **JM Tox / JM Bioanálises**.

## Comentários

- Quando visitantes deixam comentários, são coletados:
  - Dados mostrados no formulário de comentários.  
  - Endereço de IP.  
  - Dados do navegador (user agent).  
- Esses dados ajudam na **detecção de spam**.

### Gravatar

- Uma sequência anonimizada (hash) criada a partir do e-mail do usuário pode ser enviada ao serviço **Gravatar** para verificar se há cadastro.  
- Política de privacidade do Gravatar: <https://automattic.com/privacy/>  
- Após aprovação do comentário, a foto de perfil associada ao Gravatar pode ficar visível publicamente junto ao comentário.

## Mídia

- Se o usuário enviar imagens para o site, deve evitar arquivos com dados de localização incorporados (**EXIF GPS**).  
- Visitantes podem baixar imagens e extrair dados de localização a partir desses metadados.

## Mídia incorporada de outros sites

- Artigos podem incluir conteúdo incorporado (vídeos, imagens, artigos etc.).  
- Conteúdos incorporados de outros sites se comportam como se o visitante estivesse navegando diretamente nesses outros sites, podendo seguir as políticas de privacidade próprias deles.

## Com quem compartilhamos seus dados

- Se o usuário solicitar redefinição de senha, o endereço de IP poderá ser incluído no e-mail de redefinição.

## Por quanto tempo mantemos os seus dados

- Comentários e metadados associados são mantidos **indefinidamente**, para reconhecer e aprovar automaticamente comentários posteriores, em vez de retê-los para moderação.  
- Para usuários registrados (se houver registro no site):
  - São guardadas as informações pessoais fornecidas no perfil de usuário.  
  - Todos os usuários podem ver, editar ou excluir suas informações pessoais a qualquer momento (exceto o **username**, que não pode ser alterado).  
  - Administradores de sites podem ver e editar essas informações.

## Quais são os seus direitos sobre seus dados

Texto base:

- Usuários com conta no site ou que deixaram comentários podem solicitar:
  - Um arquivo exportado com os dados pessoais armazenados, incluindo quaisquer dados fornecidos.  
  - A remoção dos dados pessoais mantidos sobre si.

Limitações:

- A remoção não inclui dados que o site seja obrigado a manter para fins **administrativos, legais ou de segurança**.

## Para onde enviamos seus dados

- Comentários de visitantes podem ser verificados por um serviço automático de detecção de spam.
